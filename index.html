Privacy Policy

LinkedIn Exporter
Last updated: June 15, 2026
Developer: Cheeznology

Overview

LinkedIn Exporter is a Chrome extension that reads profile information from LinkedIn pages you are already viewing and lets you copy or export that data on your device. There is no backend server. We do not sell user data.

Data we access

When you open the extension on a LinkedIn profile page (linkedin.com/in/...), it may read visible page content including:

  • Full name
  • Headline
  • Location
  • Current company
  • Profile URL
  • Profile photo URL

The extension only runs when you interact with it on a page you have open. It does not browse LinkedIn in the background.

Data we store

The following is stored locally in your browser using chrome.storage.local:

  • Export history (up to 50 profiles)
  • Multi-profile queue (up to 100 entries)
  • Tags and notes you add
  • Export timestamps

This data stays on your device. The developer does not receive it.

Data we do not collect

  • Passwords or LinkedIn login credentials
  • Messages or personal communications
  • Browsing history outside the active profile tab you use with the extension
  • Keystrokes, clicks, or mouse movement
  • Data sent to external servers
  • Remote code execution

Permissions

Permission Why it is used
storage Save export history, queue, tags, and notes locally
activeTab Access the current tab when you open the popup
https://www.linkedin.com/* Read visible profile fields on LinkedIn profile pages

Data sharing

We do not sell, rent, or transfer your data to third parties.

Children's privacy

This extension is not directed at children under 13.

Changes

This policy may be updated if the extension changes. The date at the top will reflect any revision.
